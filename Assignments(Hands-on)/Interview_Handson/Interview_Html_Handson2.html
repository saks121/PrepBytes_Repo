HTML Handson Day 2

1.Diffrence Between div and span tag.

Div:It's a block element that the whole space ,it is defined/created in. By Default ,A Space and '/n' is applied.

Span: It's a inline element that takes the space of a line ,in which it is defined. No '/n' or Space is applied.

2.Diffrence Between class and id tag.

class:it represents a group of elements having same properties.we can apply this to multiple elements and style them in one go.

Id: it represents a unique element by it's identity which is nothing but a random name ,we give it to.Using this we can apply styling on it.

3.What are the self close tags

Tags that are closed by putting a '/' at end and don't require closing tag. Example-img,br,hr,etc.

4.How many types of heading are there? How to decide which to use where.

There are 6 tags in headings.i.e h1,h2,h3,h4,h5,h6. h1 is used for main heading and h2,h3,h4...upto h6 used for subheading. we can use the tags as per our requirement.

5.Diffrence between inline and blockline.

inline- inline elements do not force a new line to begin in the document flow. block- on other hand ,typically cause a line break to occur.starts with new line.

6.Diffrence between em and strong.

strong- used to indicate and highlight textsto the browser

em - used to make text emphasize.While reading webpage using special voice tools, the text(emphasized) comes in a diffreent way.

7.What are attributes?

Attributes give the additional information about the tag.It should written in lowercase. multiple attributes are used up in tags.
